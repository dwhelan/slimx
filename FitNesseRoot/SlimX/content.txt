!3 Classes
The following tables allow you to access and inspect classes:

|[[Import Classes][SupportTables.ImportClasses]]|Allows you to specify variable classes without a package name in your tables. You can also remove imported classes and show all imported classes.|
|[[Show Class][SupportTables.ShowClass]]|Shows properties, constructors and methods of a class. Useful when authoring acceptance tests so you can see what properties, functions and constructors are available.|
!3 Variables
The following tables allow you to create, manipulate, check and show variables:

|[[Create Variable][SupportTables.CreateVariable]]|Creates variables of arbitrary classes.|
|[[Show Variable][SupportTables.ShowVariable]]|Shows properties of variables.|
|[[Access Variable][SupportTables.ShowVariable]]|Provides access to variables for calling functions and checking.|
|[[Check Variable][SupportTables.CheckVariable]]|Check the properties of a variable.|
|[[Create Check Variable][SupportTables.CreateCheckVariable]]|Creates the wiki text for a [[Check Variable][SupportTables.CheckVariable]] of a variable from a [[Show Variable][SupportTables.ShowVariable]] table. Used to convert exploratory tests into regression tests|
!3 Properties
Properties are obtained from the variables following the normal Java ''getter'' convention.
For example, every variable has a ''class'' property because every Java object has a ''getClass()'' method.
!3 Functions
The [[Create Variable][SupportTables.CreateVariable]] and [[Access Variable][SupportTables.ShowVariable]] script tables set the System Under Test object to the specified variable so that you can invoke any public method on a variable.
