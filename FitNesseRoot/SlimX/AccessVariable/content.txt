This [[script table][http://fitnesse.org/FitNesse.UserGuide.SliM.ScriptTable]] allows you to access variables.

When you run this table simply sets the !-SystemUnderTest-! to be the provided variable so you can use standard script table features to access it.

|script|access variable|$foo|
|ensure|equals|$foo|
|show|get field 1|
|check|to string|foo, 0.0|
